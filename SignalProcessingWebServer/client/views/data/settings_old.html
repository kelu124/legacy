The number of saved points per lines. This number is a power of two (by exemple, if you set 3, you will have 2³ images, which means 8 pixels per lines)

The Decimation is the number of ignored points you have between two saved points. This number will be a power of 8 (if you set it to 1, you will have 8¹ points between two saved points, which means 8 points between two saved points). The values of each saved points are averaged with the ignored ones
